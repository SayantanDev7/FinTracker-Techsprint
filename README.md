FinTracker is a **next-generation finance tracking web app** that allows users to monitor their **cryptocurrency and stock investments** in real-time. Built for speed, simplicity, and secure authentication, FinTracker combines modern **frontend UI**, **real-time API data**, and **interactive charts** to give users a seamless experience.

---


##  Features

- **Secure Authentication**: Sign up & login with **Firebase Auth** (Email/Password + OAuth2).  
- **Live Crypto & Stock Tracking**: Fetch prices and historical trends using **CoinGecko API**.  
- **Interactive Charts**: Dynamic portfolio visualization with **Chart.js**.  
- **Responsive UI**: Mobile-first design using **Tailwind CSS**.  
- **Vanilla JS Logic**: Smooth interactivity without heavy frameworks.  
- **Node.js Backend**: Handles API requests securely.

---


## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | HTML, CSS, Vanilla JS, Tailwind CSS, Chart.js |
| Backend | Node.js |
| Authentication | Firebase Auth, OAuth2 |
| APIs | CoinGecko API |
